## 준비물

| 준비물 | 설명 |
|--------|------|
| Raspberry Pi Pico 2 WH | Wi-Fi가 내장된 마이크로컨트롤러예요. |
| Micro USB **데이터** 케이블 | 충전 전용은 안 돼요! 데이터 전송이 가능한 케이블이 필요해요. |
| Thonny IDE | MicroPython 코드를 작성하고 실행하는 프로그램이에요. |
| AI 채팅 환경 | Claude, ChatGPT 등 — 바이브 코딩에 사용해요. |

<Aside type="caution" title="Micro USB 데이터 케이블 확인">
Pico 2 WH는 **Micro USB** 포트를 사용합니다 (USB-C가 아닙니다!). 반드시 **데이터 전송이 가능한** Micro USB 케이블을 사용하세요. 충전 전용 케이블로는 Thonny에서 Pico가 인식되지 않습니다.
</Aside>

## Step 1: wifi_config.py 저장

두 개의 파일을 Pico에 저장할 거예요. 먼저 Wi-Fi 정보가 들어있는 `wifi_config.py`부터 시작해요.

<Aside type="tip" title="Thonny에서 Pico에 파일 저장하는 방법">
1. Thonny 메뉴 → 파일 → 다른 이름으로 저장
2. "어디에 저장할까요?" → **"Raspberry Pi Pico"** 선택
3. 파일 이름 입력 (`wifi_config.py`)
4. 저장!
</Aside>

<div class="code-file-label">wifi_config.py — Pico에 저장</div>

```python
# wifi_config.py
# Pico에 저장해두면 다른 코드에서 import해서 쓸 수 있어요
WIFI_SSID = "교실Wi-Fi이름"
WIFI_PASSWORD = "비밀번호"
```

<Tabs>
  <TabItem label="🏫 학교에서">
    학교 Wi-Fi는 **비밀번호가 설정**되어 있어요. 선생님이 SSID와 비밀번호를 칠판에 적어줄 거예요. Pico는 **2.4GHz만 지원**하므로 5GHz 전용 네트워크는 사용할 수 없어요.

    ```python
    # wifi_config.py — 학교 버전
    WIFI_SSID = "학교Wi-Fi이름"        # 선생님이 알려준 SSID
    WIFI_PASSWORD = "학교Wi-Fi비밀번호"  # 선생님이 알려준 비밀번호
    ```

    <Aside type="tip" title="IP를 고정하고 싶다면">
    DHCP가 자동으로 IP를 바꾸기 때문에, 매번 IP가 달라질 수 있어요. 아래처럼 **정적 IP**를 설정하면 항상 같은 주소로 접속할 수 있습니다.

    ```python
    # wifi_config.py — 학교 정적 IP 버전
    WIFI_SSID = "학교Wi-Fi이름"
    WIFI_PASSWORD = "학교Wi-Fi비밀번호"

    STATIC_IP = "172.30.6.10"    # 다른 기기와 안 겹치는 번호
    SUBNET    = "255.255.0.0"    # 학교 네트워크는 보통 /16 대역
    GATEWAY   = "172.30.0.1"     # 기본 게이트웨이
    DNS       = "8.8.8.8"
    ```

    `main.py`에서 Wi-Fi 연결 직후에 한 줄 추가:

    ```python
    wlan.connect(WIFI_SSID, WIFI_PASSWORD)
    while not wlan.isconnected():
        time.sleep(1)
    # ↓ 이 줄을 추가하면 IP 고정!
    wlan.ifconfig((STATIC_IP, SUBNET, GATEWAY, DNS))
    ```
    </Aside>

    **학교 IP 대역 확인법:** Thonny 셸에서 연결 후 `wlan.ifconfig()`를 실행하면 자동 할당된 IP, 서브넷, 게이트웨이를 확인할 수 있어요. 정적 IP를 설정할 때 이 값들을 참고하세요.
  </TabItem>
  <TabItem label="🏠 집에서">
    집에 가져가서 실습하려면 `wifi_config.py`만 바꾸면 돼요! 집 공유기의 **2.4GHz SSID**를 확인하세요 (5GHz와 이름이 다를 수 있어요).

    **1단계: 기본 연결 (이것만 하면 바로 동작!)**

    ```python
    # wifi_config.py — 집 버전
    WIFI_SSID = "우리집Wi-Fi이름"
    WIFI_PASSWORD = "우리집비밀번호"
    ```

    이렇게만 바꾸고 실행하면 Pico가 자동으로 IP를 받아요. Thonny 셸에 뜨는 IP로 접속하면 끝!

    **2단계 (선택): 내 네트워크 정보 확인하기**

    정적 IP를 설정하려면 먼저 집 네트워크 정보를 알아야 해요. **가장 쉬운 방법**은 Pico를 먼저 연결한 뒤 Thonny 셸에서 확인하는 거예요:

    ```python
    # Thonny 셸에서 실행 (Pico가 Wi-Fi 연결된 상태에서)
    >>> wlan.ifconfig()
    ('192.168.0.15', '255.255.255.0', '192.168.0.1', '168.126.63.1')
    #   ↑ 내 IP          ↑ 서브넷           ↑ 게이트웨이       ↑ DNS
    ```

    <Aside type="caution" title="Pico에 접속이 안 될 때">
    노트북/스마트폰이 Pico와 **다른 네트워크**에 연결되어 있으면 접속할 수 없어요. 예를 들어 Pico는 `192.168.0.x`인데 스마트폰이 `172.30.x.x`(학교 Wi-Fi)에 연결되어 있으면 서브넷이 달라서 접속 불가! 둘 다 **같은 Wi-Fi**에 연결되어 있는지 확인하세요.
    </Aside>

    공유기마다 대역이 다르므로 **반드시 직접 확인**하세요:

    | 공유기 | 흔한 대역 | 서브넷 |
    |--------|----------|--------|
    | iptime | `192.168.0.x` | `255.255.255.0` |
    | KT/SKT/LG | `172.30.x.x` 또는 `192.168.x.x` | 다양함 |
    | TP-Link, ASUS | `192.168.0.x` 또는 `192.168.1.x` | `255.255.255.0` |

    Pico 없이 PC/스마트폰에서 확인하는 방법:
    - **Windows:** `Win+R` → `cmd` → `ipconfig` 입력 → **IPv4 주소**, **서브넷 마스크**, **기본 게이트웨이** 확인
    - **Mac:** Option(⌥) 키를 누른 채 메뉴바 Wi-Fi 아이콘 클릭 → **IP 주소**, **서브넷 마스크**, **라우터** 표시됨
    - **iPhone:** 설정 → Wi-Fi → 연결된 네트워크 옆 ⓘ 버튼 → **IP 주소**, **서브넷 마스크**, **라우터**
    - **Android:** 설정 → Wi-Fi → 연결된 네트워크 길게 누르기 → 네트워크 세부정보

    **3단계 (선택): 정적 IP 설정**

    2단계에서 확인한 값을 기반으로 설정합니다:

    ```python
    # wifi_config.py — 집 정적 IP 버전
    # (아래 숫자들은 예시! 반드시 2단계에서 확인한 값으로 바꾸세요)
    WIFI_SSID = "우리집Wi-Fi이름"
    WIFI_PASSWORD = "우리집비밀번호"

    STATIC_IP = "192.168.0.200"    # 내 IP의 마지막 숫자만 200~250으로 변경
    SUBNET    = "255.255.255.0"    # 서브넷은 그대로 복사
    GATEWAY   = "192.168.0.1"      # 게이트웨이도 그대로 복사
    DNS       = "8.8.8.8"          # 구글 DNS (고정값)
    ```

    <Aside type="tip" title="STATIC_IP는 어떻게 정하나요?">
    `wlan.ifconfig()`로 확인한 **내 IP의 마지막 숫자만 바꾸면** 돼요. 예를 들어 자동 할당이 `192.168.0.15`였다면, 정적 IP는 `192.168.0.200`처럼 **200~250 사이** 높은 번호를 쓰세요. 이 범위는 보통 다른 기기가 쓰지 않아요. 나머지 값(서브넷, 게이트웨이)은 확인한 그대로 복사합니다.
    </Aside>

    `main.py`는 수정할 필요 없어요! `wifi_config.py`만 교체하면 어디서든 동일하게 작동합니다.
  </TabItem>
</Tabs>

```python collapse={10-181}
# ============================================
# 교실 Wi-Fi 모니터 - 1차시 기본 코드
# Pico 2 WH 웹서버 + 자기소개 페이지
# ============================================

import network
import socket
import time
from wifi_config import WIFI_SSID, WIFI_PASSWORD

# ── Wi-Fi 연결 ──────────────────────────────
wlan = network.WLAN(network.STA_IF)
wlan.active(True)

print("Wi-Fi 연결 중...")
wlan.connect(WIFI_SSID, WIFI_PASSWORD)

wait = 0
while not wlan.isconnected():
    print(f"  대기 중... ({wait}초)")
    time.sleep(1)
    wait += 1
    if wait > 15:
        print("연결 실패! wifi_config.py를 확인하세요.")
        raise SystemExit

my_ip = wlan.ifconfig()[0]
rssi = wlan.status('rssi')
print(f"연결 완료! IP: {my_ip} / RSSI: {rssi} dBm")

# ── HTML 페이지 ─────────────────────────────
def make_page():
    rssi = wlan.status('rssi')
    uptime = time.ticks_ms() // 1000

    return f"""<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pico 모니터</title>
    <style>
        * {{ margin: 0; padding: 0; box-sizing: border-box; }}
        body {{
            font-family: -apple-system, 'Segoe UI', sans-serif;
            background: #0f0f23;
            color: #ccc;
            padding: 20px;
            min-height: 100vh;
        }}
        .header {{
            text-align: center;
            padding: 20px 0;
        }}
        .header h1 {{
            color: #00d4ff;
            font-size: 1.6em;
            margin-bottom: 5px;
        }}
        .header p {{
            color: #666;
            font-size: 0.9em;
        }}
        .grid {{
            display: grid;
            grid-template-columns: 1fr 1fr;
            gap: 12px;
            max-width: 500px;
            margin: 20px auto;
        }}
        .card {{
            background: #1a1a3e;
            border-radius: 12px;
            padding: 18px;
            text-align: center;
            border: 1px solid #2a2a5e;
        }}
        .card.wide {{
            grid-column: span 2;
        }}
        .card .icon {{
            font-size: 1.8em;
            margin-bottom: 8px;
        }}
        .card .label {{
            color: #888;
            font-size: 0.8em;
            margin-bottom: 4px;
        }}
        .card .value {{
            color: #00d4ff;
            font-size: 1.4em;
            font-weight: bold;
        }}
        .footer {{
            text-align: center;
            margin-top: 30px;
            color: #444;
            font-size: 0.8em;
        }}
    </style>
</head>
<body>
    <div class="header">
        <h1>Pico Wi-Fi 모니터</h1>
        <p>Raspberry Pi Pico 2 WH</p>
    </div>

    <div class="grid">
        <div class="card">
            <div class="icon">📶</div>
            <div class="label">Wi-Fi 신호</div>
            <div class="value">{rssi} dBm</div>
        </div>

        <div class="card">
            <div class="icon">🌐</div>
            <div class="label">IP 주소</div>
            <div class="value" style="font-size:1em;">{my_ip}</div>
        </div>

        <div class="card">
            <div class="icon">⏱️</div>
            <div class="label">가동 시간</div>
            <div class="value">{uptime}초</div>
        </div>

        <div class="card">
            <div class="icon">👤</div>
            <div class="label">만든 사람</div>
            <div class="value">이름</div>
        </div>

        <div class="card wide">
            <div class="label">새로고침하면 데이터가 업데이트됩니다</div>
        </div>
    </div>

    <div class="footer">
        Powered by MicroPython on Pico 2 WH
    </div>
</body>
</html>"""

# ── 웹서버 시작 ─────────────────────────────
server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
server.bind(('0.0.0.0', 80))
server.listen(2)

print()
print("=" * 45)
print(f"  웹서버 시작! http://{my_ip}")
print("  브라우저에서 접속하세요")
print("=" * 45)

while True:
    try:
        client, addr = server.accept()
        request = client.recv(1024)

        # favicon 요청 무시
        req_str = request.decode()
        if 'favicon' in req_str:
            client.close()
            continue

        print(f"접속: {addr[0]}")
        page = make_page()

        client.send(b"HTTP/1.1 200 OK\r\nContent-Type: text/html; charset=utf-8\r\nConnection: close\r\n\r\n")
        client.send(page.encode())
        client.close()

    except Exception as e:
        print(f"에러: {e}")
        try:
            client.close()
        except:
            pass
```

## Step 3: 브라우저에서 접속

Thonny에서 **실행(Run)** 버튼을 누르면 Shell에 이런 메시지가 나와요:

```
=============================================
  웹서버 시작! http://192.168.X.X
  브라우저에서 접속하세요
=============================================
```

노트북 브라우저에 저 주소를 입력하면 여러분의 첫 웹페이지가 나타나요!

<Aside type="caution" title="접속이 안 될 때 3가지 확인">
1. 노트북이 **교실 Wi-Fi**에 연결되어 있나요?
2. 주소가 `http://`인가요? (`https://`로 자동 변환되면 안 돼요!)
3. IP 주소를 정확하게 입력했나요?
</Aside>

<Aside type="note" title="스마트폰/태블릿에서도 접속 가능">
같은 Wi-Fi에 연결된 기기라면 스마트폰이나 태블릿에서도 접속할 수 있습니다. 모바일 데이터로는 접속이 안 됩니다.
</Aside>

RSSI가 뭐냐고요? **Wi-Fi 신호 세기**예요. 0에 가까울수록 강하고, -90에 가까울수록 약해요. 새로고침하면 값이 달라지는 걸 확인해보세요!

짝꿍의 IP 주소로 접속하면 짝꿍의 페이지가 뜨는 것도 시도해보세요. IP 주소만 알면 접속할 수 있어요 — 이게 인터넷의 기본 원리예요.

## 마무리

오늘 한 일을 정리해볼까요?

1. **Pico를 웹서버로 만들었어요.** 여러분 손에 있는 이 작은 보드가 네이버 서버와 같은 원리로 작동하는 거예요!
2. **바이브 코딩을 처음 해봤어요.** AI에게 요청하고, 결과를 보고, 다시 수정하는 사이클을 경험했어요.

오늘의 팁 두 가지만 기억하세요:
- **코드는 반드시 붙여넣기!** 프롬프트가 길든 짧든, 코드 없이 보내면 AI가 못 도와줘요.
- **라이트로 시작, 필요하면 구체화!** 처음부터 길게 쓸 필요 없어요. 짧게 보내보고, 결과가 아쉬우면 그때 디테일을 추가하세요.
