### 프로젝트 1: 학교 공기질 종합 모니터링 시스템

**문제:** 교실마다 공기질이 다른데, 언제 어디서 환기해야 할지 데이터가 없다.

**센서 구성:**
- DHT20 온습도 센서 (I2C)
- SGP30 공기질 센서 (I2C) -- CO2, TVOC
- 빛 센서 (ADC) -- 수업 중/쉬는 시간 구분용

**ML 활용:**
- 센서 데이터를 10분 단위로 수집하여 CSV로 저장
- "환기 필요" / "적정" / "쾌적" 3개 클래스로 라벨링
- Teachable Machine 또는 규칙 기반 분류 모델 적용
- 환기 필요 시 자동 알림 (LED + OLED 경고)

**이중 대시보드:**
- OLED: CO2 값 + 온도 + 상태 아이콘 + ML 예측 결과
- PC: matplotlib 실시간 그래프 + 시간대별 환기 추천

**장기 운영 설계:**
```python
# 파일 크기 관리
MAX_FILE_SIZE = 500_000  # 500KB
# 날짜별 파일 분리
filename = f"data_{day_count}.csv"
# 에러 카운터
if error_count > 100:
    # 센서 재초기화
    reset_sensors()
```

### 프로젝트 2: 교실 에너지 절약 자동화 시스템

**문제:** 교실 조명과 환기가 필요 없을 때도 켜져 있어 에너지가 낭비된다.

**센서 구성:**
- 빛 센서 (ADC) -- 자연광 측정
- 초음파 거리 센서 (Digital) -- 사람 존재 감지
- DHT20 온습도 센서 (I2C) -- 냉난방 효율 측정

**ML 활용:**
- "교실 사용 중" / "비어 있음" / "쉬는 시간" 3개 클래스
- 거리 센서 + 소리 + 빛 패턴으로 분류
- 비어 있으면 -> "조명/냉방 OFF 권장" 알림

**이중 대시보드:**
- OLED: 현재 상태 + 절약 가능 에너지 추정
- PC: 시간대별 에너지 절약 기회 분석 그래프

**절약 효과 추정 코드:**
```python
# 조명이 불필요하게 켜져 있던 시간 추정
if room_empty and light_on:
    waste_minutes += interval / 60
    waste_kwh = waste_minutes / 60 * 0.04  # 40W 조명 기준
    print(f"절약 가능: {waste_kwh:.3f}kWh")
```

### 프로젝트 3: 센서 기반 동작 분류 ML 시스템

**문제:** 착용형 센서로 사람의 동작(앉기/걷기/달리기)을 자동 분류할 수 있을까?

**센서 구성:**
- ADXL345 가속도 센서 (I2C) -- 3축 가속도
- DHT20 온습도 센서 (I2C) -- 체온 변화 (간접)
- 소리 센서 (ADC) -- 발소리 감지

**ML 활용:**
- 동작별 가속도 데이터를 20Hz로 수집 (10초 x 동작별 5회)
- "앉기" / "걷기" / "달리기" / "점프" 4개 클래스
- Teachable Machine에 CSV 업로드 또는 규칙 기반 분류

**이중 대시보드:**
- OLED: 실시간 X/Y/Z 그래프 + 현재 감지된 동작
- PC: 동작별 시간 분포 파이 차트

**데이터 수집 -> ML 파이프라인:**
```
1. Pico에서 가속도 데이터 CSV 수집 (동작별)
2. CSV를 PC로 전송
3. Teachable Machine에 업로드하여 학습
4. 학습된 모델로 실시간 분류
5. 분류 결과를 대시보드에 표시
```

## 시스템 설계 가이드

### 아키텍처 설계

마스터 프로젝트는 시스템 수준의 설계가 필요해요.

```
[센서 계층]          [처리 계층]         [표시 계층]
 센서 1 ─┐          ┌─ 데이터 저장      ┌─ OLED 대시보드
 센서 2 ─┼─ Pico ─┼─ ML 분류         ─┼─ PC 대시보드
 센서 3 ─┘          └─ 알림 판단        └─ LED/부저 알림
```

### 장기 운영 체크리스트

- [ ] `try/except`로 모든 센서 읽기를 감싸고 있다
- [ ] 파일 크기가 너무 커지지 않도록 관리한다
- [ ] 에러가 100회 이상이면 센서를 재초기화한다
- [ ] `main.py`로 저장하여 전원만 연결하면 자동 시작된다
