## 활동 가이드

### Step 1. 학습 여정 회고 (15분)

15차시 동안의 여정을 되돌아보세요. 아래 질문에 조용히 답을 적어보세요.

**15차시 여정 요약:**

| 막 | 핵심 전환 | 여러분의 변화 |
|----|----------|-------------|
| **1막 (1~4차시)** | 직접 코딩 | 센서를 연결하고 코드를 직접 입력하며 기초를 배웠어요 |
| **2막 (5~10차시)** | 바이브 코딩 | AI의 도움을 받아 더 복잡한 센서와 코드에 도전했어요 |
| **3막 (11~15차시)** | 대시보드 + ML | 대시보드로 데이터를 시각화하고, ML로 패턴을 분류했어요 |

**성찰 질문:**

1. **1차시를 떠올려보세요.** 처음 센서를 연결했을 때 가장 어려웠던 것은? 지금은 어떤가요?
2. **바이브 코딩을 떠올려보세요.** AI에게 코드를 요청할 때 가장 중요한 것은 무엇이었나요?
3. **대시보드를 떠올려보세요.** OLED에 데이터가 나타났을 때의 느낌은?
4. **ML을 떠올려보세요.** 컴퓨터가 소리를 분류했을 때 놀라웠나요?
5. **팀 프로젝트를 떠올려보세요.** 팀에서 나의 역할은 무엇이었고, 다음에는 어떤 역할을 해보고 싶나요?

:::note[성찰 작성 팁]
"잘했다/못했다"의 평가보다는, "무엇을 경험했는지"를 적어보세요. "대시보드가 어려웠다"보다는 "OLED에 막대그래프를 그릴 때 좌표 계산이 헷갈렸는데, AI 코드를 분석하면서 이해됐다"처럼 구체적으로 적으면 더 좋아요.
:::

#### 센서 여권

|------|----------|----------|---------------|----------------|
| 온습도 (DHT20) | 1, 3, 12차시 | 교실 온도, 습도 | I2C, while, 대시보드 | 같은 교실 안에서도 온도차가 있다 |

### 나의 성장 시각화 (PC Python)

```python
# 자기 평가 점수를 입력하고 성장 그래프를 그립니다

# 각 항목에 대해 1차시 때와 지금을 비교 (1~5점)
# 여러분의 실제 느낌을 솔직하게 입력하세요!
skills = {
    "센서 연결": {"before": 1, "after": 4},
    "코드 작성": {"before": 1, "after": 3},
    "바이브 코딩": {"before": 1, "after": 4},
    "대시보드 제작": {"before": 1, "after": 3},
    "데이터 해석": {"before": 1, "after": 4},
    "ML 이해": {"before": 1, "after": 3},
    "팀 협업": {"before": 2, "after": 4},
    "발표 능력": {"before": 2, "after": 3},
}

print("=== 나의 성장 그래프 ===\n")
print(f"{'역량':14s} | {'1차시':6s} | {'지금':6s} | 성장")
print("-" * 55)

total_growth = 0
for skill, scores in skills.items():
    before = scores["before"]
    after = scores["after"]
    growth = after - before
    total_growth += growth

    before_bar = "*" * before
    after_bar = "#" * after
    arrow = f"+{growth}" if growth > 0 else f"{growth}"

    print(f"{skill:14s} | {before_bar:6s} | {after_bar:6s} | {arrow}")

print("-" * 55)
avg_growth = total_growth / len(skills)
print(f"{'평균 성장':14s} |        |        | +{avg_growth:.1f}점")

# 성장 메시지와 코스 B 추천
print("\n=== 코스 B 추천 레벨 ===")
if avg_growth >= 3:
    print("놀라운 성장이에요! 코스 B Lv.3(연구자)부터 도전해보세요.")
elif avg_growth >= 2:
    print("꾸준히 성장했어요! 코스 B Lv.2(발명가)부터 시작해보세요.")
elif avg_growth >= 1:
    print("좋은 시작이에요! 코스 B Lv.1(탐험가)부터 시작해보세요.")
else:
    print("관심 있는 차시를 복습한 뒤 코스 B에 도전해보세요.")

# 3막에서 추가된 역량 강조
new_skills = ["바이브 코딩", "대시보드 제작", "ML 이해"]
print(f"\n=== 3막에서 새로 배운 역량 ===")
for skill in new_skills:
    s = skills[skill]
    print(f"  {skill}: {s['before']}점 -> {s['after']}점 (+{s['after']-s['before']})")
```

:::note[자기 평가 방법]
점수는 여러분이 직접 매기는 거예요. `"before"` 값은 1차시 때의 나, `"after"` 값은 지금의 나를 1~5점으로 평가하세요. "바이브 코딩", "대시보드 제작", "ML 이해"는 3막에서 새로 배운 역량이에요.
:::

## 잠깐, 15주 전의 나를 떠올려보세요

15주 전, 여러분은 아마 이런 상태였을 거예요.

- "센서가 뭐지?"
- "MicroPython? 파이썬이랑 다른 건가?"
- "`print`가 뭔지도 모르겠다"

**지금 여러분은:**
- 센서를 연결하고 코드로 데이터를 읽을 수 있어요
- AI에게 코드를 요청하고 수정할 수 있어요 (바이브 코딩)
- OLED와 PC에 실시간 대시보드를 만들 수 있어요
- Teachable Machine으로 ML 모델을 학습시킬 수 있어요
- 데이터를 그래프로 바꾸고, 발표로 전달할 수 있어요

**직접 코딩 -> 바이브 코딩 -> 대시보드 -> ML.** 이것이 15주 동안의 여정이에요. 여러분은 더 이상 코드를 "배우는 사람"이 아니에요. 코드와 AI를 **"사용하는 사람"**이 되었어요.
