# 코스 B로의 다리

> **코스 A를 마쳤다면, 이제 스스로 만들 차례예요.**

---

## 코스 A에서 배운 것

15차시 동안 여러분은 **센서로 세상을 읽고, 대시보드로 보여주고, AI로 분석하는 방법**을 배웠어요.

<Tabs>
  <TabItem label="1막: 직접 코딩">

### 1막: 직접 코딩 (1~4차시)

| 차시 | 주제 | 핵심 역량 |
|------|------|----------|
| 1차시 | 센서로 세상 읽기 | DHT20 연결, I2C, while 루프 |
| 2차시 | 빛을 숫자로 | 빛 센서, ADC, Plotter |
| 3차시 | 자동으로 기록하기 | CSV 파일 저장, 시간 기록 |
| 4차시 | 소리를 숫자로 | 소리 센서, 변수, 조건문 |

**핵심 전환:** "센서가 세상을 숫자로 바꿀 수 있다고?"

  </TabItem>
  <TabItem label="2막: 바이브 코딩">

### 2막: 바이브 코딩 (5~10차시)

| 차시 | 주제 | 핵심 역량 |
|------|------|----------|
| 5차시 | 교실 공기를 숫자로 | 공기질 센서, 환기 실험 |
| 6차시 | 거리를 재는 눈 | 초음파 센서, 함수 정의 |
| 7차시 | 움직임을 잡아라 | 가속도 센서, 3축 데이터 |
| 8차시 | 센서 데이터 다듬기 | 이동 평균, 이상치 제거 |
| 9차시 | 여러 센서 한 번에 | 복합 센서, 데이터 통합 |
| 10차시 | AI에게 코드 맡기기 | 바이브 코딩 전략, AI 활용 |

**핵심 전환:** "AI와 협업하면 더 복잡한 것도 만들 수 있어!"

  </TabItem>
  <TabItem label="3막: 대시보드와 ML">

### 3막: 대시보드와 머신러닝 (11~15차시)

| 차시 | 주제 | 핵심 역량 |
|------|------|----------|
| 11차시 | 팀 프로젝트 설계 | 문제 발견, 통합 설계, 바이브 코딩 전략 |
| 12차시 | 실시간 대시보드 | OLED + PC 이중 대시보드 |
| 13차시 | 센서 데이터로 ML | Teachable Machine, ML 파이프라인 |
| 14차시 | 발표와 피드백 | 데이터 소통, 구조화된 피드백 |
| 15차시 | 되돌아보기 | 성찰, 코드 총정리, 코스 B 안내 |

**핵심 전환:** "데이터를 대시보드로 보여주고, AI로 분석할 수 있다!"

  </TabItem>
</Tabs>

## 코스 A vs 코스 B 비교

|  | 코스 A | 코스 B |
|---|--------|--------|
| **형태** | 수업형 (교사 주도) | 프로젝트형 (학생 주도) |
| **시간** | 50분 x 15주 | 3시간 x 레벨별 |
| **구조** | 정해진 차시 순서 | 레벨 자유 선택 |
| **센서** | 차시별 지정 | 학생이 선택 |
| **코딩 방식** | 직접 코딩 -> 바이브 코딩 | 자유 선택 (직접 + 바이브 혼합) |
| **AI/ML** | Teachable Machine 체험 | ML 모델 설계 + 배포 |
| **대시보드** | OLED + PC 기초 | 종합 시스템 구축 |
| **목표** | 기초 역량 습득 | 실전 프로젝트 완성 |
| **평가** | 체크리스트 자기점검 | 산출물 기반 평가 |
| **적합 대상** | 처음 시작하는 학생 | 기초를 마친 학생 |

## 코스 B 레벨별 추천 시작점

여러분의 상태에 맞는 레벨에서 시작하세요.

### 코스 A를 모두 이수한 학생

| 자기 평가 | 추천 레벨 | 설명 |
|-----------|----------|------|
| "센서 연결은 되지만 코드가 아직 어려워요" | **Lv.1 탐험가** | 단일 센서 + OLED로 기초를 다지세요 |
| "코드도 쓸 수 있고 바이브 코딩도 익숙해요" | **Lv.2 발명가** | 복합 센서 + 스마트 시스템에 도전하세요 |
| "팀 프로젝트가 재미있었고 더 깊이 연구하고 싶어요" | **Lv.3 연구자** | 가설 기반 장기 연구를 해보세요 |
| "대시보드도 ML도 자신 있고 실제 문제를 해결하고 싶어요" | **Lv.4 마스터** | 다중 센서 + ML + 실제 문제 해결에 도전하세요 |
