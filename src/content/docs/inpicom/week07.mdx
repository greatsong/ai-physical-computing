### 연결 방법

SCD30을 **I2C1 포트**에, OLED를 **I2C0 포트**에, RGB LED Stick을 **D16 포트**에 연결하세요.

:::tip[I2C 포트 2개 사용]
오늘은 I2C 장치 2개(SCD30, OLED)를 사용합니다. 각각 다른 I2C 포트에 연결하면 주소 충돌 걱정 없이 동시에 사용할 수 있습니다. SCD30은 주소 0x61, OLED는 주소 0x3C으로 서로 다릅니다.
:::

:::note[SCD30 예열 시간]
SCD30은 전원을 켠 후 처음 몇 회 측정값이 불안정할 수 있습니다. 약 30초 정도 기다리면 안정적인 값이 나옵니다. 측정 주기는 2초입니다.
:::

## 활동 가이드

### Step 1. 연결하기 (5분)

1. **SCD30 센서를 Grove 케이블로 I2C1 포트에 연결합니다.** (GP6=SDA, GP7=SCL)
2. **OLED를 Grove 케이블로 I2C0 포트에 연결합니다.** (GP8=SDA, GP9=SCL)
3. **RGB LED Stick을 D16 포트에 연결합니다.**
4. **Micro USB 케이블로 컴퓨터와 연결합니다.**

:::note[드라이버 파일 확인]
SCD30 센서를 사용하려면 `scd30.py` 드라이버 파일이 Pico에 저장되어 있어야 합니다. OLED도 `ssd1306.py`가 필요합니다. 선생님의 안내에 따라 미리 업로드했는지 확인하세요.
:::

### Step 2. CO2 측정하기 (10분)

1. Thonny를 열고 아래 **기본 코드**를 입력합니다.
2. 실행 후 Shell에 CO2, 온도, 습도가 표시되는지 확인합니다.
3. **입김 테스트**: 센서에 "후~" 불어보세요. CO2 값이 급상승하는 것을 관찰합니다.
4. 10초 후 값이 서서히 내려오는 것을 확인합니다.

:::note[CO2 ppm 기준]
ppm은 "parts per million(백만분의 일)"입니다. 실외 깨끗한 공기는 약 400ppm, 실내 기준 상한은 1000ppm, 1500ppm이 넘으면 환기가 필요하고, 2000ppm 이상이면 집중력이 크게 저하됩니다.
:::

### Step 3. 환기 알리미 만들기 (20분)

1. **도전 코드**를 입력합니다.
2. 정상 상태에서 LED가 초록색이고 OLED에 "GOOD"이 표시되는지 확인합니다.
3. 센서에 입김을 불어 CO2를 올려봅니다. LED가 노란색 -> 빨간색으로 바뀌는지, OLED 상태가 바뀌는지 관찰합니다.
4. 임계값(`WARN`, `BAD`)을 조절하여 여러분 교실에 맞는 기준을 찾아보세요.

### Step 4. 교실 모니터링 체험 (10분)

환기 알리미를 교실 여러 곳에 놓아보며 CO2 농도를 비교하세요.

1. 창문 가까운 곳 vs 창문에서 먼 곳
2. 사람이 밀집된 곳 vs 빈 곳
3. 창문을 열기 전 vs 열고 5분 후

### 기본 코드

```python
# === SCD30 CO2 센서 읽기 ===
# CO2 농도(ppm), 온도, 습도 측정

from machine import I2C, Pin
import scd30
import time

# --- I2C 설정 (I2C1 포트) ---
i2c = I2C(1, sda=Pin(6), scl=Pin(7))

# --- SCD30 초기화 ---
sensor = scd30.SCD30(i2c, addr=0x61)
sensor.start_continous_measurement()

print("=== CO2 모니터 시작 ===")
print("예열 중... (처음 몇 회 값이 불안정할 수 있습니다)")

while True:
    if sensor.data_ready():
        co2, temp, humidity = sensor.read_measurement()
        print(f"CO2: {co2:.0f} ppm | 온도: {temp:.1f}C | 습도: {humidity:.1f}%")
    time.sleep(2)  # SCD30 측정 주기 = 2초
```

### 줄별 해설

| 줄 | 코드 | 설명 |
|----|------|------|
| 8 | `I2C(1, sda=Pin(6), scl=Pin(7))` | I2C1 포트를 초기화합니다. SCD30은 I2C 통신으로 데이터를 보냅니다. |
| 11 | `scd30.SCD30(i2c, addr=0x61)` | SCD30 센서 객체를 생성합니다. I2C 주소 0x61은 SCD30 고유 주소입니다. |
| 12 | `start_continous_measurement()` | 연속 측정 모드를 시작합니다. 2초마다 자동으로 CO2, 온도, 습도를 측정합니다. |
| 17 | `sensor.data_ready()` | 새 측정 데이터가 준비되었는지 확인합니다. 2초마다 True가 됩니다. |
| 18 | `read_measurement()` | CO2(ppm), 온도(C), 습도(%) 세 값을 한 번에 읽습니다. |

### 도전 코드: OLED 대시보드 + LED 경보

```python
# === CO2 환기 알리미 – 풀 대시보드 ===
# OLED에 수치 + LED에 색상 경보

from machine import I2C, Pin
from neopixel import NeoPixel
import ssd1306
import scd30
import time

# --- 하드웨어 설정 ---
# SCD30은 I2C1 (GP6, GP7)
i2c1 = I2C(1, sda=Pin(6), scl=Pin(7))
sensor = scd30.SCD30(i2c1, addr=0x61)
sensor.start_continous_measurement()

# OLED는 I2C0 (GP8, GP9)
i2c0 = I2C(0, sda=Pin(8), scl=Pin(9))
oled = ssd1306.SSD1306_I2C(128, 64, i2c0)

# LED Stick은 D16
NUM_LEDS = 10
np = NeoPixel(Pin(16), NUM_LEDS, timing=(280, 515, 515, 745))

# --- CO2 임계값 (ppm) ---
WARN = 1000
BAD  = 1500

def set_all(r, g, b):
    for i in range(NUM_LEDS):
        np[i] = (r, g, b)
    np.write()

def get_status(co2):
    """CO2 값에 따른 상태 판정 + LED 색상 변경"""
    if co2 < WARN:
        set_all(0, 50, 0)
        return "GOOD"
    elif co2 < BAD:
        set_all(255, 200, 0)
        return "WARN - Ventilate?"
    else:
        set_all(255, 0, 0)
        return "BAD - Open Window!"

def draw_bar(oled, co2):
    """CO2 농도를 막대 그래프로 표시"""
    bar_width = min(int(co2 / 2000 * 120), 120)
    oled.rect(4, 50, 120, 10, 1)          # 테두리
    oled.fill_rect(4, 50, bar_width, 10, 1)  # 채움

# --- 메인 루프 ---
print("=== CO2 대시보드 시작 ===")

while True:
    if sensor.data_ready():
        co2, temp, hum = sensor.read_measurement()
        status = get_status(co2)

        # OLED 갱신
        oled.fill(0)
        oled.text("CO2 Monitor", 20, 0)
        oled.text(f"CO2: {co2:.0f} ppm", 0, 14)
        oled.text(f"T:{temp:.1f}C H:{hum:.0f}%", 0, 28)
        oled.text(status, 0, 40)
        draw_bar(oled, co2)
        oled.show()

        print(f"CO2: {co2:.0f}ppm | {status}")

    time.sleep(2)
```

:::note[코드 읽어보기]
I2C 포트를 2개 사용합니다. `I2C(1, ...)`은 I2C1 포트(SCD30 연결), `I2C(0, ...)`은 I2C0 포트(OLED 연결)입니다. `draw_bar()` 함수는 CO2 농도를 120px 너비의 막대 그래프로 변환합니다. 2000ppm이 꽉 차는 기준이며, `WARN`과 `BAD` 값을 교실 상황에 맞게 바꿔보세요.
:::

## 잠깐, 방금 무슨 일이 일어난 걸까요?

여러분이 방금 한 일을 다시 생각해보세요.

교실 공기 속의 **CO2 분자**가
**->** SCD30 내부에서 적외선을 **흡수**하고 (NDIR 방식)
**->** 흡수량이 **ppm 농도**로 계산되어 I2C로 Pico에 전달되고
**->** 코드가 농도에 따라 **상태를 판정**하고 (GOOD / WARN / BAD)
**->** LED가 **색상**으로, OLED가 **숫자와 그래프**로 동시에 알려주었습니다.

"환기해야 할 것 같아"라는 주관적 감각이, ppm 단위의 객관적 데이터로 바뀌었습니다. 학교, 사무실, 카페에 설치된 공기질 모니터가 바로 이 원리입니다.
