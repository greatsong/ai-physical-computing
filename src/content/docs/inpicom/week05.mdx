## 준비물

| 준비물 | 설명 |
|--------|------|
| Raspberry Pi Pico 2 WH | MicroPython이 동작하는 마이크로컨트롤러입니다. |
| Grove Shield for Pi Pico | Pico 위에 올려 꽂는 확장 보드입니다. Grove 커넥터 포트가 여러 개 있어 센서를 케이블 하나로 연결할 수 있습니다. |
| Grove 케이블 (4핀) x 2 | 센서와 Shield를 연결하는 케이블입니다. |
| Grove mini PIR 모션 센서 | 사람 몸에서 나오는 적외선의 **변화**를 감지하는 디지털 센서입니다. 움직임이 있으면 1, 없으면 0을 출력합니다. |
| Grove WS2813 LED Strip (60 LED) | 1m에 60개 LED가 달린 풀컬러 LED 띠입니다. NeoPixel 방식으로 각 LED 색상을 개별 제어할 수 있습니다. |
| Micro USB 케이블 | Pico와 컴퓨터를 연결하여 전원을 공급하고 코드를 전송합니다. |

### 연결 방법

PIR 모션 센서를 **D18 포트**에, LED Strip을 **D16 포트**에 연결하세요.

:::tip[Grove 커넥터 안내]
4핀 커넥터라 방향이 정해져 있어서, 홈에 맞춰 끝까지 밀어 넣으면 됩니다. 잘못 꽂을 걱정이 없습니다.
:::

:::note[포트 찾기]
PIR과 LED Strip은 모두 **디지털 센서/액추에이터**입니다. "D18", "D16"이라고 적힌 포트를 찾아 연결하세요. 아날로그(A0) 포트나 I2C 포트에 꽂으면 동작하지 않습니다.
:::

## 활동 가이드

### Step 1. 연결하기 (5분)

1. **Pico 2 WH가 Grove Shield 위에 올려져 있는지 확인합니다.**
2. **PIR 모션 센서를 Grove 케이블로 D18 포트에 연결합니다.**
3. **LED Strip을 Grove 케이블로 D16 포트에 연결합니다.** Strip의 **IN** 쪽 커넥터를 사용하세요.
4. **Micro USB 케이블로 컴퓨터와 연결합니다.**

:::tip
PIR 센서 전면의 하얀 돔 모양 부분이 적외선을 감지하는 렌즈입니다. 이 면이 사람 쪽을 향하도록 놓아야 합니다.
:::

### Step 2. PIR 모션 감지 테스트 (10분)

1. Thonny를 열고 아래 **기본 코드**를 입력합니다.
2. **실행(Run)** 버튼을 클릭합니다.
3. 센서 앞에서 손을 흔들어보세요. "감지!"가 출력되나요?
4. 이번에는 **가만히 서 있어보세요.** 감지가 멈추는 것을 확인합니다.

:::note[생각해보기]
PIR은 **움직임**을 감지하는 센서입니다. 적외선의 **변화**를 읽기 때문에, 가만히 있으면 감지하지 못합니다. 이것이 실제 센서등이 가끔 꺼지는 이유입니다. 감지 범위는 약 2~3m, 각도는 약 100도입니다.
:::

### Step 3. 스마트 조명 만들기 (20분)

1. 아래 **도전 코드**를 입력합니다.
2. 실행한 후 센서 앞에서 움직여보세요. LED가 서서히 밝아지는지 확인합니다.
3. 가만히 있으면 10초 후 서서히 꺼지는지 확인합니다.
4. `AUTO_OFF_SEC` 값을 5초, 20초로 바꿔보며 적절한 시간을 찾아보세요.

### Step 4. 에너지 절감 토론 (10분)

이 시스템을 학교 복도에 설치한다고 가정하고, 모둠에서 이야기해보세요.

1. **"복도 불이 8시간 내내 켜져 있으면 몇 Wh를 쓸까?"** (40W 기준)
2. **"센서등으로 바꾸면 실제 점등 시간이 얼마나 줄어들까?"**
3. **"PIR 센서의 한계(가만히 있으면 꺼짐)를 보완할 방법은?"**

### 기본 코드

```python
# === PIR 모션 센서 기본 테스트 ===
# 움직임이 감지되면 "감지!" 출력

from machine import Pin
import time

# --- PIR 센서 설정 ---
# D18 포트 = GP18, 디지털 입력
pir = Pin(18, Pin.IN)

# --- 감지 루프 ---
print("=== 모션 감지 시작 ===")
print("센서 앞에서 움직여보세요!")

count = 0  # 감지 횟수

while True:
    if pir.value() == 1:
        count += 1
        print(f"감지! (#{count})")
        time.sleep(2)   # PIR은 감지 후 잠시 유지됨
    else:
        print("... 대기 중")
    time.sleep(0.5)
```

### 줄별 해설

| 줄 | 코드 | 설명 |
|----|------|------|
| 8 | `Pin(18, Pin.IN)` | GP18 핀을 디지털 입력으로 설정합니다. PIR 센서는 0(대기) 또는 1(감지)만 출력하는 디지털 센서입니다. |
| 17 | `pir.value() == 1` | 센서 값을 읽어 1이면 움직임이 감지된 것입니다. ADC처럼 0~65535 범위가 아니라, 딱 0 아니면 1입니다. |
| 19 | `time.sleep(2)` | PIR은 한 번 감지하면 신호가 2~3초간 유지됩니다. 이 대기 없이 읽으면 한 번의 움직임에 여러 번 "감지!"가 출력됩니다. |

### 도전 코드: 페이드 인/아웃 스마트 조명

```python
# === 스마트 조명 – 부드러운 점등/소등 ===
# 페이드 인/아웃 + 자동 소등 타이머

from machine import Pin
from neopixel import NeoPixel
import time

# --- 하드웨어 설정 ---
pir = Pin(18, Pin.IN)
NUM_LEDS = 60
np = NeoPixel(Pin(16), NUM_LEDS)

# --- 설정값 ---
AUTO_OFF_SEC = 10   # 감지 없으면 10초 후 소등
WARM_WHITE = (255, 200, 100)  # 전구색

# --- 부드러운 전환 함수 ---
def fade_in(steps=20):
    """서서히 밝아지기"""
    for s in range(1, steps + 1):
        brightness = s / steps
        r = int(WARM_WHITE[0] * brightness)
        g = int(WARM_WHITE[1] * brightness)
        b = int(WARM_WHITE[2] * brightness)
        for i in range(NUM_LEDS):
            np[i] = (r, g, b)
        np.write()
        time.sleep(0.03)

def fade_out(steps=20):
    """서서히 어두워지기"""
    for s in range(steps, 0, -1):
        brightness = s / steps
        r = int(WARM_WHITE[0] * brightness)
        g = int(WARM_WHITE[1] * brightness)
        b = int(WARM_WHITE[2] * brightness)
        for i in range(NUM_LEDS):
            np[i] = (r, g, b)
        np.write()
        time.sleep(0.03)
    # 완전 소등
    for i in range(NUM_LEDS):
        np[i] = (0, 0, 0)
    np.write()

# --- 메인 루프 ---
print("=== 스마트 조명 시작 ===")
is_on = False
last_detect = time.ticks_ms()

while True:
    if pir.value() == 1:
        last_detect = time.ticks_ms()  # 감지 시각 갱신
        if not is_on:
            print("감지 -> 페이드 인!")
            fade_in()
            is_on = True

    # 자동 소등: 마지막 감지 후 N초 경과
    elapsed = time.ticks_diff(time.ticks_ms(), last_detect)
    if is_on and elapsed > AUTO_OFF_SEC * 1000:
        print("타임아웃 -> 페이드 아웃")
        fade_out()
        is_on = False

    time.sleep(0.2)
```

:::note[코드 읽어보기]
`time.ticks_ms()`는 Pico가 시작된 이후 흘러간 시간을 밀리초(ms)로 돌려줍니다. `ticks_diff()`로 두 시점의 차이를 구하면 "마지막 감지 이후 얼마나 지났는지" 계산할 수 있습니다. `AUTO_OFF_SEC`를 바꿔서 자동 소등 시간을 조절해보세요.
:::

## 잠깐, 방금 무슨 일이 일어난 걸까요?

여러분이 방금 한 일을 다시 생각해보세요.

사람 몸에서 나오는 **적외선**이
**->** PIR 센서 내부의 초전소자에서 **전기 신호**로 바뀌고
**->** 그 신호가 디지털 값(**0 또는 1**)으로 Pico에 전달되고
**->** 코드가 "1이면 LED 켜기, 일정 시간 지나면 끄기"라는 **판단**을 하고
**->** NeoPixel LED Strip이 **부드럽게 점등/소등**되었습니다.

아파트 복도의 센서등, 화장실 자동 조명, 현관의 자동 점등 시스템 -- 모두 이 원리로 동작합니다. 여러분은 방금 그 원리를 직접 구현한 것입니다.

## 마무리

### 오늘 배운 것

- **PIR 모션 센서**: 적외선의 변화를 감지하는 디지털 센서를 사용했습니다. 출력은 0(대기) 또는 1(감지)로 아주 단순합니다.
- **자동 소등 타이머**: `time.ticks_ms()`로 시간을 측정하고, 일정 시간이 지나면 자동으로 LED를 끄는 로직을 만들었습니다.
- **페이드 효과**: 밝기를 단계적으로 바꿔 서서히 켜지고 꺼지는 부드러운 전환 효과를 구현했습니다.
